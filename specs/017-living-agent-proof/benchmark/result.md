# Controlled Living-Agent Dogfood Result

## Protocol

Two fresh Codex sessions received the same Nutrition Planning brief, the same three local images,
the same output format, and no access to parent project files.

- **Control:** brief only.
- **Treatment:** brief plus four accumulated DESIGN:OS lesson groups covering orchestration,
  imagery, section depth, layout diversity, icon policy, motion, and coherence.
- A separate session scored anonymous desktop and mobile renders.
- Round-one curator feedback was then reapplied only to treatment and judged by a fresh curator.

Anonymous mapping: A = treatment, B = control.

## Results

| Metric | Control | Treatment | Delta |
|---|---:|---:|---:|
| Blind curator score, final | 89 | **94** | **+5** |
| Art direction | 13 | **14** | +1 |
| Section depth | 14 | **15** | +1 |
| Layout diversity | 13 | **14** | +1 |
| Image use | 9 | **10** | +1 |
| Coherence | 17 | **18** | +1 |
| Prohibited text/Unicode interface glyphs | 27 | **0** | **-100%** |
| Scroll-reveal implementation | none | present | treatment only |
| Reduced-motion handling | present | present | no regression |
| Repair rounds observed | 0 | 1 | treatment used one learning iteration |

Treatment moved from 89 to 94 after applying the first blind review. The second blind evaluator
classified it above the declared 90-point world-class threshold and named exceptional lower-page
investment, purposeful layout variety, meaningful imagery, and mobile recomposition.

## Proof verdict

The deterministic proof remains **APPLIED**, not **IMPROVING**.

Why it does not graduate:

- The declared causal bar requires a +10 curator delta; the observed delta is +5.
- Treatment required one feedback iteration while the control was not repaired.
- Contradiction and cross-project isolation runs have not yet passed.
- One controlled case cannot establish a longitudinal trend.

What is proven:

- Accumulated lessons changed first-delivery behavior.
- A previously repeated hard-rule category fell from 27 violations to 0.
- Applying independent feedback improved the treatment from 89 to 94 without user repetition.
- The final treatment crossed the world-class visual threshold in a blind review.

## Remaining visible weaknesses

The final curator still noted a dense hero, some small mobile copy, and horizontally clipped
weekday navigation. These remain product learning inputs; the artifact is evidence, not a claim of
perfection.
