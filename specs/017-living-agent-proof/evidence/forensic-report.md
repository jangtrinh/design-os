# Living Agent Forensic Evidence

## Verdict

The three projects support different levels of the proof ladder. Together they establish that
DESIGN:OS can persist identity, turn feedback into later decisions, and convert escaped defect
classes into executable prevention. They do not yet establish controlled memory-caused creative
improvement.

| Project | Supported level | Evidence | Blocker to next level |
|---|---|---|---|
| ease-design | APPLIED | User feedback changed later orchestration, and a memory-on treatment beat its memory-off control 94 to 89 | Comparison delta stayed below +10; contradiction and isolation runs are missing |
| VSF-PCP | LEARNING | Ratified soul, 168 events, four harvested insights | No applied-memory receipt; all insights seen once |
| platform-design-system | APPLIED | Escaped defects became G6/G8/G9, negative fixtures, and later detections | No controlled creative comparison or safeguards |

## Platform prevention result

- Gate suite v1.0 to v1.4.
- Negative self-tests 14 to 36.
- Four curator defect classes converted into deterministic checks.
- New gates later detected defects beyond the curator sample.
- Owner-caught defects reported as 7 before the suite and 0 after it.
- Final audit reported 36/36 self-tests and `overallPass: true`.

## Infrastructure findings

- VSF-PCP contains unresolved designer-correction records.
- platform-design-system contains competing root and nested memory stores.
- platform-design-system has heartbeat configuration without heartbeat state.
- Neither external project has recurrent insights (`seen > 1`).
- Neither project records a formal memory retrieval and application receipt.

## Honest claim now supported

DESIGN:OS has demonstrated persistent project identity, feedback-to-decision application, and a
project-specific verification system that learned from escaped defects and prevented silent
recurrence. A controlled creative comparison favored the learning-enabled workflow, but it did
not clear the predeclared causal threshold, so sustained improvement remains unproven.
